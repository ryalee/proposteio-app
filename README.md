### Proposteio -> Gerador de propostas com IA para freelancers!

Atuo como freelancer, e como todo bom profissional autônomo eu sempre tive uma dificuldade em obter retorno quando enviava propostas (principalmente no 99freelas que era minha principal), nunca sabia muito bem como estruturar uma boa proposta... foi aí que decidi criar uma ferramenta que me ajudasse. 

Sei que muito provavelmente existe alguma por aí que poderia dar conta do recado mas preferi criar a minha própria pois é um projetinho a mais para meu portfólio :)
Feito com NextJS, Typescript, Tailwind, API do Groq (uma ia gratuita e bem interessante) 

Testei algumas vezes e... não é que o negócio deu bom? kkkkkkkkk
